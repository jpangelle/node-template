# node-template

Template to make a node server

## Features

- TypeScript
- Linting
- Prettier
- Pre-commit hook
- env ready

### Development

#### Clone the repo

```sh
git clone git@github.com:<github-username>/<repo-name>.git
```

#### Install dependencies

```sh
yarn install
```

#### Create environment variable file

```sh
touch .env
```

#### Run app in development mode

```sh
yarn dev
```

#### Run linter

```sh
yarn lint
```

### Production

#### Build app

```sh
yarn build
```

#### Run in production mode

```sh
yarn start
```
